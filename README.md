
# Pokédex Web App

A responsive Pokédex web application built with React that allows users to browse, search, and explore Pokémon data using the PokéAPI.
This project focuses on clean component structure, efficient data fetching, and user-friendly UI interactions. 
## Live Demo

https://whitebangul.github.io/Pokedex/


## Tech Stack

**Frontend**: React (Hooks)

**Styling**: CSS

**API**: PokéAPI

**Language**: JavaScript (ES6+)
## Features

* Search Pokemon by name
* Filter Pokemon by type
* View Detailed Pokemon information:
    * Types
    * Abilities
    * Base stats
    * Evolution tree
* Client-side caching to reduce redundant  API calls
* Modal-based detail view for better UX
* Loading & error handling states

## Future Improvements
* Favorite Pokémon feature (localStorage)
* Mobile-first responsive design
* Multi-language support (EN / KO toggle)
